# (BFSエントリ)モバイル_サービスサマリ_端末

- マート名: `b_hjn_bfs_モバイル_サービスサマリ_端末`
- CSVファイル名: 
  - 初期データ: `b_hjn_bfs_モバイル_サービスサマリ_端末.csv`

- 参考データ: `sample_data/bfs_service_summary_devices.csv`
- 補足: 参考CSVは502列のため、ExcelシートのうちCSV未採用の末尾2項目は記載していません。

# (Mars)取次店

- マート名: `m_取次店_all`
- CSVファイル名:
  - 初期データ: `m_取次店_all.csv`
  - 差分データ: `m_取次店_all_diff.csv`
- 全量更新 or 差分更新: 差分更新
- 更新頻度: 日次
- データ数: 初期移行（120万件）、差分更新（53件）
- データ量: 初期移行（300MB）、差分更新（80KB）
- データ概要: 全ての代理店（親代理店及び、子代理店）情報を保有。 店舗で扱う商材についてソフトバンクや一次代理店等への発注、ソフトバンクからの請求を行う業務から「受注先」「請求先」という説明を記載しております。 M_取次店 :M_取次店_ALLから最新フラグが「1」のデータのみ抽出（前日断面情報、有効開始日が当日を除く） 有効終了日が1年以内の情報
